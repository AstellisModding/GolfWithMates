# Golf With Mates

Minecraft Mod. Feature rich minecraft golfing mod. For survival or creative games.

Place a course, grab a club, and sink the ball. now with ball physics.

---


## Clubs

| Club | Shot style |
|------|-----------|
| Putter | Flat roll, short range |
| Iron | Shallow arc, mid range |
| Wedge | High arc, short range |
| Driver | Flat drive, long range |

Use **Scroll + Shift** to adjust power before swinging.
