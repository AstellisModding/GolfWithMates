# Golf With Mates

A multiplayer golf mod for Minecraft 1.21.1 on NeoForge.

Place a course, grab a club, and sink the ball. Physics runs server-side and animates client-side.

---

## Requirements

- Minecraft 1.21.1
- [NeoForge](https://neoforged.net/) 21.1.190+

## Install

Drop the `.jar` into your `mods/` folder and launch.

---

## Clubs

| Club | Shot style |
|------|-----------|
| Putter | Flat roll, short range |
| Iron | Shallow arc, mid range |
| Wedge | High arc, short range |
| Driver | Flat drive, long range |

Use **Scroll + Shift** to adjust power before swinging.
